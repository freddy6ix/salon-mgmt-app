## Legend

- **Blue** = working (TIME IN / TIME OUT)
- **Gray** = day off
- `SPLIT` = Milano split shift (two separate blocks in one day)
- `*` = reading is approximate; exact boundary hard to determine from screenshots

## Schedule Convention

In the app database, `day_of_week` uses ISO numbering: **0 = Monday … 6 = Sunday**.

| DB value | Day |
|----------|-----|
| 0 | Monday |
| 1 | Tuesday |
| 2 | Wednesday |
| 3 | Thursday |
| 4 | Friday |
| 5 | Saturday |
| 6 | Sunday |
