# Salon Lyol Management System

Cloud-based salon management software for **Salon Lyol** (Toronto), designed to replace Milano software and eventually serve other salons as a multi-tenant SaaS.

## Status

**Phase 1 — in design.** Core appointment book + client management for Salon Lyol. End-to-end ERM under construction.

- [`docs/domain/`](./docs/domain) — business rules and domain model
- [`docs/erm/`](./docs/erm) — entity relationship model (in progress)
- [`docs/adr/`](./docs/adr) — architecture decision records

## Roadmap

| Phase | Scope |
|-------|-------|
| 1 | Core appointment book + client management, Salon Lyol only |
| 2 | POS integration, reporting, CRM AI (email + chat) |
| 3 | Multi-tenancy hardening + beta salons onboarded |
| 4 | Voice AI + advanced features |
